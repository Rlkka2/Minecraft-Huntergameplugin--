## ✨ 插件特性 /pom.xml
- 支持 **猎人 / 逃亡者** 双阵营游戏模式
- 完整游戏流程控制：开始、停止
- 玩家死亡、退出自动判断游戏胜负
- 指令系统简洁易用
- 无第三方依赖，纯原生 Paper API
- 适配 **Java 21 + Paper 1.21.8**

---

## 🧩 环境要求
- 服务端：**Paper 1.21.8**
- Java 版本：**JDK 21**
- 插件加载方式：放入 `plugins` 文件夹

---

## 📦 安装方法
- 将插件 jar 放入服务器 `plugins` 目录
- 启动服务器，自动生成配置文件
-  按需修改 `config.yml`
- 使用指令 `/hunter` 开始游戏

---

## 🕹 可用指令 /resources/plugin.yml
| 指令 | 功能 |
|------|------|
| `/hg caeate [房间名]` | 创建游戏房间 |
| `/hg start [房间名]` | 强制开始指定游戏房间 {admin}|
| `/hg leave [房间名]` | 退出指定游戏房间 |
| `/hg delete [房间名]` | 删除指定房间 |
| `/hg list` | 列出所有游戏房间 |
| `/fh [玩家名]` | 使用钻石复活指定幸存者 |
| `/ggbo` | 发起投降 |
